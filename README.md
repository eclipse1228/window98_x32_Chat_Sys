# window98_x32_chat_sys  
**windows98 theme webSocket chatSystem.**    
Start to   
```js
node chat.js 
```   
## Work
1. 접속 path 설정하기 - 고병수
2. 채팅창 크기 늘리기 - 정주영
3. 채팅 엔터로 입력되게 하기 - 박소원
4. name 정의하기 -> 채팅창에 상대방 이름 정하기. -남현식
6. CSS -> 정주영 : 기본적인
